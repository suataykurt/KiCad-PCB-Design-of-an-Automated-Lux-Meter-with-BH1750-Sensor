# KiCad-PCB-Design-of-an-Automated-Lux-Meter

This repository contains the PCB design, schematic, and 3D visualization of a digital Lux Meter project, developed using KiCad. The system utilizes the **BH1750** ambient light sensor and an **I2C LCD** display for real-time light intensity monitoring.

## 📋 Project Overview
The project involves the complete design cycle for a light-measuring device, from schematic entry to PCB layout and 3D visualization. It demonstrates how to integrate I2C-based sensors with an Arduino-based controller to create a compact, functional measurement tool.

## 🛠 Design Highlights
* **Schematic Design:** Integration of the BH1750 light sensor, I2C LCD interface, and Arduino Nano controller.
* **PCB Layout:** Custom-designed PCB layout in KiCad, focusing on signal integrity and component placement.
* **3D Visualization:** In-depth examination of the 3D model for physical form factor and footprint validation.
* **Power Management:** On-board voltage regulation (AMS1117-5.0) ensuring stable operation.

## 🚀 Key Features
* **BH1750 Sensor Integration:** High-precision ambient light measurement.
* **Interactive Interface:** I2C LCD for clear, real-time data display.
* **Professional CAD Standards:** Industry-standard PCB design practices.

1. Clone the repository and open the project using **KiCad**.
2. Explore the PCB layout to view routing and component placement.
3. Use the KiCad 3D Viewer to inspect the hardware assembly models.
